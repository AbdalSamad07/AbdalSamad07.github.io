Programming in Java - Spaceship Game

This game is the final project that I have completed within the first year of studying at City University. This project demonstrates all types of functions that I have learnt throughout my year.The Spaceship consists of 5 levels with the rocket being the controllable character. Each level contains different rockets that get updates upon completeing the level. The player is controller through "w" and "s" to move and they can use the "spacebar" to shoot lasers. I have implemented multiple collision methods such as Enemy Ships and Asteroids. This will be the enemies within the game. The player has 5 lives that get reset upon reaching other levels. I have also added a save and load state method that allows players to come back to their current level and progress throughout the game.
